Automatisch neue Spiele für Ihre Seitenleiste!

== Description ==

arkavis.de bietet Ihnen eine Möglichkeit ganz einfach hunderte von Spielen in Ihr Wordpress zu integrieren 
und automatisch neue Spiele zu beziehen.

Inklusive:
* Kommentare
* Bewertungen
* Sharing über Facebook, StudiVZ, Twitter und Email
(nur wenn Spiele-Ansicht POP_UP_BIG oder IN_BROWSER_POP_UP_BIG)

Einstellungen:
* Breite und Höhe
* Anzahl angezeigter Spiele (auf einmal und gesamt)
* Spiele-Ansicht als PopUp oder InBrowserPopUp
* Spielauswahl nach neueste/meist gespielteste/best bewerteste Spiele
* Farb- und erweiterte Spielauswahl (Kategorien) über arkavis.de

Autor URL: http://www.arkavis.de
Plugin URL: http://www.arkavis.de/wordpress

== Installation ==

1. zip nach wp-content/plugins entpacken
2. arkavis sidebar plugin aktivieren unter Plugins
3. gegebenfalls Einstellungen->arkavis Seitenleiste anpassen
4. Design->Widgets: arkavis Spiele für Seitenleisten in Widget-Bereich ziehen

Weitere Einstellungen:
Auf arkavis.de kann man mittels Assistent eine individuelles SiteId erstellen (Widget erzeugen und SiteId kopieren).
Damit kann man ein eigenes Farbschema designen und die Spieleauswahl bestimmen.

Außerdem gibt es auf arkavis.de weitere Widgets zur Integration in Ihr Wordpress.
